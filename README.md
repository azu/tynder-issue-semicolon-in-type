# tynder-issue-semicolon-in-type

## Install


    yarn install

## Usage

    yarn run build

Result:

```
$ tynder compile --indir schema --outdir generated
parse error occured at position:32  interfaceDefInner: Unexpected token has appeared.
     tring;
          ^~~~~~~~
    b: number;
}
```
